# Users Management and Tas Management App

## Sacks:
- React Js
- Node Js
- Typescript
- Tailwind

## Installation
#### 1. CD to the server directory and inside it run
```
$ npm install
```
```
$ npm run dev
```


#### 2. CD to the client directory and inside it run
```
$ npm install  
```
```
$ npm run start
```
